Форматы файлов

.Doc или .doc (от английского. document ) - расширение файл , используемое пдля файлов, придеоставляющихтекст , с разметкой или без. Расширение .DOC часто используется для обозначени просты текстовых файов без форматирования, однако позже стало использоваться для двоичных форматов с разметкой.
Документ MS Word

Portable Document Format (PDF) — межплатформенный открытый формат электронных документов, разработанный фирмой Adobe Systems с использованием ряда возможностей языка PostScript.
Документ PDF

ZIP — формат архивации файлов и сжатия данных без потерь. Файл в этом формате обычно имеет рсщирение .zip и хранит в сжатом или несжатом виде один или несколько файлов, которые можно из него извлечь путем распаковки с помощю сепциалной программы.
